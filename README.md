## Description

## Requirements for the working environment
You need to have an installed Node.js version of at least ***17.6***.
You can check the Node.js version with the command

```sh
$ node --version
```

## Installation

```sh
$ npm install gendiff
```

## Launch

```sh
$ gendiff _path1_ _path2_
```
